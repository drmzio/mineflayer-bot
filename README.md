# Mineflayer Bot

See [official Mineflayer documentation](https://github.com/PrismarineJS/mineflayer) for more info.

## Sample .env

```env
MC_HOST=play.example.com
MC_AUTH=microsoft
MC_USERNAME=john@example.com
MC_PASSWORD=password123
```

## Installation

Install dependencies
```bash
npm install
```

Then run the bot
```bash
npm start
```
